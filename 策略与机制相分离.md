# 机制（基层）

实现某一功能的具体执行机构



# 策略（高层）

借助于参数与算法对该功能的优化



分离的好处

可复用性：

X 成为一个通用的图形引擎，并将有关用户界面风格的决定留给工具包和其他级别的系统。我们通过指出政策和机制倾向于在不同的时间尺度上发生变异来证明这一点，政策的变化比机制快得多，GUI 工具包的外观和感觉上的时尚可能来来去去，但光栅操作和合成是永恒的。
